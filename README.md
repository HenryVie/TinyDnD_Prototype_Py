# TinyDnD_Prototype_Py
A terminal-based mini DnD-style RPG written in Python.
This is a prototype project focused on turn-based combat, dice mechanics, and simple storytelling.

You play as Jason, a warrior who accidentally unleashed demons upon the world, and now has to fix his own mess.
